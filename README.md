# Checkers Engine

A simple engine for russian checkers.

Uses minimax position scoring and alpha-beta pruning with some heuristics.

### Dependencies:
1. C/C++ compiler (C++ 17 or greater)

### Build / RUN
1. `mkdir build && cd build`
2. `cmake ..`
3. `make all`
4. `./checkers`
